- ⏰ Real-time business hours status

## How to Deploy on GitHub Pages

1. Create a new repository on GitHub
2. Upload both `index.html` and `README.md` files
3. Go to Settings → Pages
4. Under "Branch", select `main` and save
5. Your site will be published at `https://YOUR_USERNAME.github.io/REPO_NAME/`

## Customization

### Change Business Hours
Edit the `BUSINESS_HOURS` object in the JavaScript section.

### Update Phone Number for Payments
Change `PAYMENT_PHONE` variable value.

### Add/Remove Menu Items
Edit the `CATEGORIES` object structure.

### Change Images
Update URLs in the `IMAGES` object.
